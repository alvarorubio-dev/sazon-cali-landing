# 🍲 Sazón Cali - Lo mejor de la cocina caleña

Sazón Cali es una plataforma web moderna diseñada para un restaurante de alta cocina tradicional en Cali, Colombia. El sitio fusiona la herencia culinaria del Valle del Cauca con una experiencia de usuario contemporánea, permitiendo reservas en línea y visualización del menú digital.

---

## ✨ Características Principales

* **Menú Digital Interactivo:** Exhibición de platos estrella como Lomo al Trapiche y Cazuela de Mariscos con integración de pedidos vía WhatsApp.
* **Sistema de Reservas:** Formulario dinámico para reserva de mesas con confirmación en menos de 30 minutos.
* **Diseño Responsive:** Optimizado para dispositivos móviles, facilitando la navegación de comensales en movimiento.
* **Galería de Experiencias:** Sección visual que resalta los detalles gourmet y el ambiente del restaurante.
* **Integración con Google Maps:** Localización exacta en el icónico Barrio Granada, Cali.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue desarrollado utilizando las siguientes herramientas:

* **HTML5 & CSS3:** Estructura y estilos personalizados.
* **JavaScript:** Lógica para el formulario de reservas y animaciones.
